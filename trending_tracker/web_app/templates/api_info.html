Github Trending Tracker (GTT) provides a few API that are available to the public

Github Trending Pages

Endpoint: /trending/some_language_code,language code see two indices below
Parameter: since (daily, weekly, and monthly)

Example: /trending/?since=monthly
Example: /trending/c?since=weekly
Example: /trending/python?since=daily

Page index by name: for human, and for computer
Page index by time: for human, and for computer

GTT Search -- See what repos / authors has been on the Trending

Endpoint: /search/
Parameters: r (repo name), a (author/ repo owner), and type (e.g. json).

Example: /search/?a=microsoft
Example: /search/?r=rust
Example: /search/?a=microsoft&type=json
Example: /search/?r=rust&type=json
Example: /search/?a=microsoft&r=calculator&type=json

GTT Record -- Use it if you have the exact repo name and repo owner's name (how? see above)

Endpoint: /record/
Parameters: r (repo name), a (author/ repo owner), and type (e.g. json). Both r and a are required

Example: /record/?a=microsoft&r=calculator
Example: /record/?a=microsoft&r=calculator&type=json
